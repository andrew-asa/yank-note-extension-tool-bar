# yank note 工具栏

## 安装&使用
1. 安装 =》工具/扩展管理
2. 启用、关闭 =》 工具/启用工具栏
3. 选中要操作的文本进行操作
4. **注意** 如果是背景，颜色大小等一定要选择完'<span>xxx</span>'标签再进行修改
